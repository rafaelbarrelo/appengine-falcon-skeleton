Falcon Skeleton for Google App Engine (Python 2.7)
===============

Steps:

* 1 - Run backend/venv/venv.sh
* 2 - Open backend/appengine/app.yaml and set up application-id
* 3 - Run run.sh to run server
* 4 - Run deploy.sh to deploy
